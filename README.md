# DHL Assist

Proyecto web y móvil para la administración de tareas operativas, checklist, notas, alertas y módulo Departure para un entorno logístico.

## Estructura del proyecto

- backend: API, conexión a base de datos y lógica del sistema.
- frontend-web: aplicación web en React.
- frontend-mobile: aplicación móvil en React Native.
- shared: código compartido entre web y móvil.
- database: scripts de base de datos.
- docs: documentación del proyecto.
